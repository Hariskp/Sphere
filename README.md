# Sphere
Setup Sphere
1.	cd to sphere (cd Sphere)
Command : cd Sphere
2.	Dependencies Installation 
Command : npm install
3.	Installing dependencies
install dependencies 
  -	axios
  -	nodemon
  -	react
  -	react-dom
  -	react-router-dom
  -	react-slick
  -	slick-carousel
  -	sweetalert2
4.	Start sphere
Command : npm run dev
