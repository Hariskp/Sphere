# Sphere
Setup Sphere
1.	cd to sphere (cd Sphere)
Command : cd Sphere
2.	Dependencies Installation 
Command : npm install
3.	Installing dependencies
ติดตั้ง dependencies ดังนี้
  o	axios
  o	nodemon
  o	react
  o	react-dom
  o	react-router-dom
  o	react-slick
  o	slick-carousel
  o	sweetalert2
4.	Start sphere
Command : npm run dev
